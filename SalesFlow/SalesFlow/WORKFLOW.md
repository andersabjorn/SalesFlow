# Agile Workflow

## Sprint Structure
- Sprint length: 2 weeks
- Sprint 1: Lead Management (Jan 31 - Feb 14)
- Sprint 2: Offer Management (Feb 15 - Feb 28)
- Sprint 3: Order Management (Mar 1 - Mar 14)

## Definition of Done
- [ ] Code written and tested locally
- [ ] Questions answered (documented in issue)
- [ ] Code reviewed (self-review checklist)
- [ ] Committed with descriptive message
- [ ] Issue moved to Done

## Daily Practice
- One issue at a time
- Understand before moving forward
- Document learnings in issue comments
- Update project board daily
