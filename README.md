# SalesFlow

ERP module for managing sales workflow from lead to payment.

## Business Flow

Lead → Offer → Order → Invoice → Payment

## Tech Stack

- .NET 9
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Clean Architecture

## Architecture

- **Core**: Business entities, interfaces, business logic
- **Infrastructure**: Data access, external services
- **Web**: MVC controllers, views, presentation logic
